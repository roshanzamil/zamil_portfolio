# **App Name**: Roshan.dev

## Core Features:

- Home Section: Introduce Roshan and showcase key skills.
- About Me: Detail Roshan's background, experience, and personal interests relevant to web development.
- Projects Section: Display Roshan's web development projects with descriptions, links, and visuals.
- Skills Section: List technical skills with proficiency levels (e.g., HTML, CSS, JavaScript, React, Node.js).
- Contact Form: Enable visitors to send Roshan messages directly through the portfolio.
- AI project suggestion: LLM that functions as a tool; based on Roshan's skill, suggests side projects to add to the portfolio.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and expertise. This color is strong without being loud.
- Background color: Very light Lavender (#F0E6EF) to offer a soft contrast to the deep indigo. It's a visually related color, and the dark scheme creates an understated sophistication appropriate for a professional presence.
- Accent color: Violet (#8F00FF) for interactive elements, call-to-action buttons, and highlights to add a pop of color without being distracting.
- Body and headline font: 'Inter', sans-serif. A modern sans-serif that's readable.
- Use minimalist, professional icons to represent skills and technologies.
- Design a clean, responsive layout that adapts to different screen sizes.
- Incorporate subtle animations on scroll and hover to enhance user engagement.
